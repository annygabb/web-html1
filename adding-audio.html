Algumas Músicas Legais

Para Adicionar o Áudio:
Usamos a tag <audio> para incluir vários arquivos de áudio.

O atributo controls exibe os controles para tocar, pausar e ajustar o volume.
O atributo src define o caminho do arquivo de áudio.
Alternativamente, podemos usar <source> para definir diferentes formatos de áudio e garantir compatibilidade com todos os navegadores.
