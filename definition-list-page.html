HTML
HyperText Markup Language - Lingaugem de Marcação de Hipertexto usada para estruturar páginas da web.
CSS
Cascading Style Sheets - Folhas de Estilo em Cascata utilizadas para estilizar elementos HMTL.
JavaScript
Linguagem de programação que permite criar interatividade e dinamismo em páginas web.
