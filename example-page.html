Exemplo de um Elemento HTML

O elemento <p> é usado para definir parágrafos em HTML. Veja o exemplo abaixo:

        
            <p>Este é um parágrado em HTML.</p>

O navegador renderiza esse código como um parágrafo de texto da página.

Só umas observações:

  • <code> → Indica ao navegador que o conteúdo dentro dela é um código.
  • <pre> → Mantém a formatação original do texto no código.
