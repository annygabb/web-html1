Compreendendo o Espaço em Branco

O HTML ignora múltiplos espaços em branco e os reduz a um único espaço.

Podemos usar a tag <br>para criar quebras de linha:
Esta é a primeira linha
Esta é a segunda linha

Esta a terceira linha.

Se adicionarmos vários espaços seguidos, como este: teste, eles serão reduzidos a um único espaço.

Para preservar espaços e quebras de linha exatamente como foram escritos, podemos usar a tag <pre>:

    Este   é   um   exemplo   com   múltipos   espaços.
    Cada quebra de linha será mantida.
