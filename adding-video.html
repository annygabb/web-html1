Vídeo de Jujutsu Kaisen

Sukuna Vs Mahoraga

Explicação:
Usamos a tag <video> para adicionar vídeos ao HTML.
O atributo controls exibe os controles de play, pause e volume.
O atributo src define o caminho do arquivo de vídeo.
O elemento <source> permite adicionar diferentes formatos de vídeo para compatibilidade com navegadores.
A mensagem dentro da tag <video> aparece se o navegador não suportar vídeos.
