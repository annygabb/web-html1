Explicação dos estilos aplicados:

  • border-collapse: collapse; → Remove espaços entre as bordas das células.
  • background-color: #f8f9fa; → Define a cor de fundo da tabela.
  • border: 2px solid black; → Adiciona bordas pretas de 2px às células.
  • padding: 10px; → Adiciona espaçamento interno às células.
  • text-align: left; → Alinha o texto à esquerda dentro das células.
  • background-color: #007bff; → Define o fundo azul para os cabeçalhos da tabela.
  • color: white; → Define a cor do texto dos cabeçalhos como branco.
  • tr:nth-child(even) → Alterna a cor de fundo para linhas pares da tabela.
